# 🔗 Code First API

### Simple RESTful Web API with ASP.NET Core & Entity Framework Core

A simple RESTful API project built using ASP.NET Core Web API, Entity Framework Core, SQL Server and Swagger. The project demonstrates Code First approach, database migration and basic CRUD API operations.

# 🌐 API Endpoints

| Method | Endpoint               | Description          |
| ------ | ---------------------- | -------------------- |
| GET    | `/api/Categories`      | Lists all categories |
| GET    | `/api/Categories/{id}` | Gets category by id  |
| POST   | `/api/Categories`      | Adds new category    |
| PUT    | `/api/Categories/{id}` | Updates category     |
| DELETE | `/api/Categories/{id}` | Deletes category     |
